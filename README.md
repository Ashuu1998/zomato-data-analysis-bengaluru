#  Zomato-data-analysis-bengaluru 

##  Project Overview

This project presents an end-to-end data analysis of the Bengaluru restaurant market using Zomato dataset. The objective was to identify the key factors influencing restaurant ratings and translate data insights into a strategic blueprint for launching a successful restaurant in a competitive urban market.

The project covers data cleaning, exploratory data analysis (EDA), visualization, and dashboard development to convert raw data into actionable business insights.


##  Business Problem

The Bengaluru restaurant industry is highly competitive, with thousands of establishments operating across different price segments, cuisines, and locations. Many new restaurants fail due to assumption-based decision-making rather than data-driven strategy.

This project answers key business questions:

- Does higher pricing lead to higher ratings?
- Which restaurant types perform better?
- Do online ordering and table booking impact ratings?
- Which locations host the most successful restaurants?
- What pricing strategy works best?

##  Data Cleaning & Preparation

- Cleaned the `rate` column by removing "/5" and handling "NEW" values
- Converted `approx_cost` from string format to numeric
- Handled missing values using statistical methods
- Removed highly null or irrelevant columns
- Standardized categorical variables

##  Exploratory Data Analysis

Analysis performed:

- Rating vs Cost relationship
- Restaurant Type vs Average Rating
- Impact of Online Ordering and Table Booking
- Location-wise restaurant distribution
- Cuisine performance comparison
- Top-performing areas identification

##  Key Insights

- Convenience features (online ordering & table booking) strongly correlate with higher ratings
- Higher cost does not guarantee higher ratings
- Mid-range pricing (₹1200–₹1500 for two) shows the highest rating density
- Specialized formats (Cafes, Dessert Bars) outperform generic multi-cuisine restaurants
- Premium commercial areas consistently host higher-rated establishments

## Final Recommendation

Based on the analysis, the ideal strategy for a new restaurant in Bengaluru would be:

- Launch a specialized Cafe or Dessert-focused concept
- Select a premium commercial location
- Maintain a mid-range pricing strategy
- Offer both online ordering and table booking services
